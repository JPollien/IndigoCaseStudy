This API provides these endpoints:

urlmethodendpointaction
/dataGETdata_listRetrieve a collection of data
/dataPOSTdata_listCreate new data
/data/<int:id>GETdata_detailRetrieve a specific record
/data/<int:id>PATCHdata_detailUpdate a specific record
/data/<int:id>DELETEdata_detailDelete a specific record

This API supports simple filters per flask-rest-jsonapi. Formatting for calls follows:

GET /data?filter[CROP]=RICE
Accepts: json


You can also include more than one simple filter in a request:

GET /data?filter[CROP]=RICE&filter[COUNTY_NAME]=PRAIRIE
Accepts: json
